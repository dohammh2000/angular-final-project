TO INSTALL THE REQUIRED MODULES FOR APP:

npm install


TO RUN PROJECT :
ng serve




TO RUN THE JSON API SERVER USE THE COMMAND : SO THE PRODUCT APPEARS
json-server --watch db.json

(db.json : IT IS THE FILE THAT HAVE PRODUTS AND USERS FROM FAKE STORE API)
